## 💻 Como Rodar o Projeto

1.  Clone o repositório: `git clone `
2.  Instale as dependências: `npm install`
3.  Inicie o servidor de desenvolvimento:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```  
5.  Abra o navegador e acesse `http://localhost:3000`
